requests-scitokens
##################

.. toctree::
   :hidden:

   Home <self>

`requests-scitokens` provides an authentication plugin for
`Requests <http://requests.readthedocs.io/>`__ that handles attaching
`SciTokens <https://scitokens.org>`__ to HTTP requests.
